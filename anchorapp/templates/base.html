Problem 1 - Divisibility

Print all integers ai such that ai is divisible by x and not divisible by y, where 1 < ai < n < 100000.
Input: First, you are given t (t < 100) - the number of test cases. In each of the following t lines, 3 integers: n x y. You might assume also that x < n and x is not divisible by y.
Output: In each of the following t lines, numbers requested in the problem description in the separated by a single space in ascending order. Print all integers ai such that ai is divisible by x and not divisible by y, where 1 < ai < n < 100000.
